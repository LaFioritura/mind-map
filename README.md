# Mind Map Aion

Visualizzazione interattiva della formazione cognitiva umana.

## Avvio locale

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Nota per Vercel

Questo progetto usa Vite + React e richiede anche `@vitejs/plugin-react`.
La dipendenza è già inclusa in `package.json`.
